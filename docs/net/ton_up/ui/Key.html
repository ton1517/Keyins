KeyConstants
 
Packagenet.ton_up.ui
Classpublic class Key
InheritanceKey Object

キーコード値を定数としてまとめたクラスです.

できるだけKeyboardクラスを使用することをおすすめします.環境によってはキーコード値が異なる可能性があります.

※記号類のキーコード値はWindowsとMacで異なる可能性が高いため現在対応していません.
